# [zEncoder](https://eliasfaisal.github.io/zEncoder/Page%20File/)
Simple Text Converting tool.
### Supported Methods
- zEncode (Default)
- Attbash
- CharCode
- Binary
- URI Component
- Hex
- Yolious Gaiser
### Usage
- The Page is located at /zEncoder/Page File
- Javascript Lib is at /zEncoder/Lib
